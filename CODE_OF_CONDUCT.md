# Code of Conduct

One rule, two clauses.

**Attack the work as hard as you can. Treat the people decently.**

- Adversarial review is the purpose here. Harsh, specific, relentless criticism of the *construction* is welcome and wanted.
- Personal attacks, harassment, and credentials-based dismissal ("an amateur can't have done this" / "a computer did this so it's worthless") are off-topic noise and will be removed. Only the locus matters.

Maintainers may edit or remove comments that violate the second clause, and may close issues that contain no checkable locus (see `CONTRIBUTING.md`). Disagreements about the physics are resolved by checking the physics, never by appeal to authority on either side.
