# What this file is — a one-page honest summary

**The differentiator is honesty, not a discovery.** Most "theory of everything" documents overclaim: they announce that nature has been explained and bury the assumptions. This one does the opposite. It states, line by line, exactly what it *assumes* versus what it *derives*, and it was put through adversarial review whose job was to find the holes — and the holes it found are printed as results, not hidden. If you have ever wanted a physics-adjacent document that tells you where it is weak before you have to dig, this is that.

## The two things it actually offers

**1. A method for honest AI-assisted research.** This is the broad, portable part, and it is a *method*, not a physics result. The rules: count zero "wins" unless something genuinely closes (it calls this PROMOTIONS: 0); never reverse-engineer a number to hit its known target, and test predictions *blind* before seeing the answer; keep the object under study frozen and read-only so it can't be edited to fit a conclusion; and run adversarial self-review that records negatives against your own interest. The file keeps two embarrassments permanently on the record — a baryon-asymmetry number that was "true by construction" (worthless), and a Cabibbo-angle prediction that was flatly falsified — as proof the discipline bites.

**2. A worked, verifiable foundations payload on one speculative geometry** (the "Fable 13D branch"). The honest core here is an *accounting* of what this particular geometry forces and what it doesn't: a no-go showing the electroweak hierarchy can't be derived inside the model; a reframing of why reality is "granular" that honestly relocates the assumption into a single named posit rather than deriving it; and a map of exactly which axioms must be assumed. A few results are correctly-imported published mathematics (re-checked given the model's matter content), each reported with its qualifier attached.

## What it is NOT — read this before sharing

- **It is not a Theory of Everything.** It says so itself.
- **There is no new tested physics, no new prediction, no new measured number.** Nothing here is falsifiable-and-confirmed.
- **The geometry is *selected*, not derived.** It was chosen as the most complete candidate by an Occam-style funnel — choosing the best survivor is explicitly not the same as proving it forced.
- **The geometry itself is unpublished and unreviewed,** living in a private corpus. Most of the file's headline counts and objects can't be confirmed from the file alone.
- **The audience is narrow:** foundations-of-quantum-mechanics / operationalist theorists, and people interested in disciplined AI-research method. For a general reader it is a methodology demonstrator, not a result.

## How to check it yourself

Three arguments in the file are finite, hand-runnable exercises — and the file names them as the only independently checkable ones: (i) a small group-theory table (the Z₆ "kernel" computation — recompute electric charge Q = T₃ + Y for each particle and check the six elements close); (ii) a short countermodel showing finitely many tests can't tell apart an infinite set of records; (iii) an interval-packing inequality. Everything else is either correctly-cited published math or an assertion about the unpublished corpus — treat those as claims, not facts.

Best way to judge it: download the file, hand it to your own AI alongside the companion prompt (`DOWNLOAD_PROMPT.md`), and tell the AI to attack it. The file explicitly does not predict its own acquittal.
