# Worked examples — does the geometry give the right numbers?

These are short, mostly non-technical notes. Each one takes the single thirteen-dimensional geometry from the main papers — a shape built only to explain the particles — shrinks it down to the four dimensions we live in, and checks it against something physics already knows cold. Every one is a **reproduce-known** check: it lands on the textbook number, and it carries the same honest caveat in its own words — *matching known physics is a hurdle every serious theory has to clear; it earns a closer look, it does not prove the bigger idea.*

| Note | Reproduces | Result |
|---|---|---|
| [The Sun as a Lens](The_Sun_as_a_Lens.pdf) | Einstein's gravitational light-bending | exactly 1.75″ at the solar limb (γ = 1, inside the Cassini bound) |
| [Hydrogen as a Quantum Electrostatic Check](Hydrogen_Quantum_Electrostatic_Check.pdf) | the hydrogen spectrum | E = −13.6 eV/n², Bohr radius a₀ ≈ 0.529 Å |
| [Single-Graviton Exchange](Single_Graviton_Quantum_Gravity_Check.pdf) | low-energy quantum gravity | Newton's 1/r potential, exactly |
| [Muon Decay](Muon_Decay_Weak_Force_Check.pdf) | the weak (Fermi) force | Γ = G_F² m_μ⁵/(192π³), term for term |
| [One Geometry, Four Forces](One_Geometry_Four_Forces.pdf) | all four force laws | each in its correct textbook form, from one shape |
| [Why Only These Particles?](Why_Only_These_Particles.pdf) | the chiral family count | exactly three, consistent with LEP N_ν = 2.984 ± 0.008 |
| [The Three-Qubit Bit-Flip Code](Bit_Flip_Code_Quantum_Computing_Check.pdf) | the QEC logical error rate | 3p² − 2p³ (no geometric input — a consistency floor) |
| [The Road in Front of the Ship](The_Road_in_Front_of_the_Ship.pdf) | a toy GR metric (speculative) | identical local proper-distance deficit — **not a drive, not faster-than-light, not hardware** |

A reproduce-known check returns the same number with or without the geometry — it is load-bearing for **consistency**, never for **correctness**, and it replaces neither Einstein nor Maxwell. The web versions, with the full reader on-ramp, are at <https://physics.magflowmeters.com/> under "How to test it."

The last note (*The Road in Front of the Ship*) is explicitly speculative — a toy metric and a fail-closed screening story, not a working drive, faster-than-light travel, propulsion, an energy source, or a Mars travel-time result.
